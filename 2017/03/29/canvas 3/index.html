canvas

  1. 逗号表达式和函数
  • 逗号分隔的表达式，会从左到右逐个执行，并且以最后一个结果为准；
  1. 将函数转换为表达式
  • 函数立即调用的语法结构：
  1. !! 变量 的作用是让param转换为布尔值
  2. 二进制运算（按位 与 或 非 异或）
  3. 两个定时器，按顺序执行，如果第一个报错，第二个定时器还会执行

    实例和原型

  • 实例产生之后，再修改构造函数的原型，对实例中的数据没有影响，但是会影响instanceof的判断

canvas

  1. 文本
    1. strokeText(); 参数：文本内容，横坐标，纵坐标，最大宽度
    2. fillText(); 参数：同上
    3. 设置文本的字体样式 ctx.font=”30px 隶书”;
    4. 对齐方式
      • 水平对齐 ctx.textAlign=”start”; //start,end,left,center,right
      • 垂直对齐 ctx.textBaseline=”top”;//top,middle,bottom,alphabehic
  • . 都有自己独立路径的方法
    1. strokeRect();
    2. fillRect();
    3. strokeText();
    4. fillText();
  1. 图片
    1. 创建图片对象 var img=new Image( );
    2. 设置图片路径 img.src=”img/boy.gif”
    3. img.onload=function(){}; 图片加载完再添加到画布中
    4. ctx.drawImage(img,图片上的坐标,图片截取的大小，画布上的坐标，画布上显示的大小)；
  2. 变换
    1. 平移 ctx.translate(坐标)
    2. 旋转 ctx.rotate(旋转的角度)[顺时针]【必须在画之前旋转】
    3. 缩放 ctx.scale(倍数，倍数);
