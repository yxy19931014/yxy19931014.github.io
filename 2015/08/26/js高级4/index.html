正则表达式 regular expression

  1. 正则的规则
    1. 元字符：描述字符的字符
      • . 单个字符（除了换行和结束符）
      • \w 单词字符
      • \s 空白符
      • [^abc] 不包括a,b,c
      • 转义： [ ]中的不需要转义
      • 如果正则直接写特定字符，就直接匹配特定字符
      • 量词： +、* 、？(0或1次) { n,m},{n} {n,}
    2. 元数据：描述数据的数据
  2. 相关的API
    1. 正则的API
      • test()
      • exec()
    2. 字符串API
      • search
      • match
      • split
      • replace
    3. str的match方法，在非全局匹配模式下，str.match(reg) 和 reg.exe(str)的作用等效。
    4. reg的exec(),可以循环遍历
