矩形的绘制

  1. 方法
    1. 逐个划线
    2. ctx.rect(横坐标，纵坐标，宽，高) 描绘路径
    3. ctx.strokeRect(横坐标，纵坐标，宽，高)有自己独自的路径，不需要显式的进行描边

      清空指定区域的画布

  2. ctx.clearRect(横坐标，纵坐标，宽度，高度);

    圆弧的绘制

  3. ctx.arc(圆心横坐标，圆心纵坐标，半径，起始角度，绘制的弧度，是否是逆时针)；最后的默认是false
  4. 一度等于180/pi ,angleToRadian(角度)将角度转换为弧度
  5. 绘制连续的路径，如果前面的终点和后面的起点一样的话就可以省略moveTo

  6. save 是用栈的方式存储，后进先出，可以保存多个状态
  7. restore()是从栈中取出样式，调用多次则是逐个取出
