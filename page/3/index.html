BOM

BOM介绍

  • 页面中的顶级对象document
  • 浏览器中的顶级对象window
  • 浏览器包括多个页面，还有其他
  • 页面中的变量和函数，document都是window的
  • 但是有些东西只能document操作，有些只能window操作
  • window在页面中可以省略

    BOM下的方法（window对象中的方法）

  1. window.open(“01祝福墙.html”,”_blank”,”width=300 height=400 resizable=no”);
    • 此方法的第一个参数是地址（可以使用外部的地址，也可以使用本地的地址）
    • 第二个参数是打开方式
      *　第三个参数是打开浏览器的样式还有其他属性
    • 返回值是一个窗口对象（win）
      1. win.close();关闭新打开的窗口
      2. win.moveTo(300,500);让新窗口移动到指定的位置
      3. win.moveBy();每次让新窗口移动多少像素
      4. win.resizeTo(400,500);让新窗口大小改变
      5. win.resizeBy(40,50);让新窗口每次变化多少
  2. location是window下的操作地址栏的对象
    • window.location.href=”http://baidu.com";//可以跳转页面，同时内容可以保存到浏览器的历史记录中，所以浏览器可以前进后退
    • window.location.assign(“http://www.baidu.com");//assign()方法可以跳转，保存历史记录
    • window.location.reload();//刷新
    • window.location.replace(“http://www.baidu.com");//replace()替换地址栏上的地址，不会保存到历史记录中
  3. navigator页面加载后获取客户端浏览器的部分信息
    • window.navigator.userAgent;//客户端浏览器的部分信息
    • window.navigator.platform;//系统信息
  4. history对象操作浏览器的历史记录
    • window.history.forward();//前进
    • window.history.back();//后退
  5. 页面加载的事件
    • window.onload=function(){
      //页面上的内容加载完才能触发
      }
    • window.onunload=function(){
      //页面关闭后触发
      }
    • window.onbeforeunload=function(){
      //页面卸载前触发（页面关闭前触发）
      }

      定时器

  6. setInterval() 每隔多少秒就执行一次代码
    • 有两个参数（时间，执行函数）

  • 美女时钟

  • 机械时钟

  1. setTimeout() 页面加载后多长时间后执行

弹性布局

  1. 开启弹性
    • 给父元素设置display: flex;子元素就可以应用弹性布局
    • 弹性布局开启之后，默认分为两个轴
      • 主轴：默认指的是从左向右—x方向
      • 副轴：默认指的是从上往下—y方向
  2. 设置主轴的对其方式
    • justify-content: space-between;元素之间会预留间隙，但是两边顶格；
    • justify-content: flex-end; 主轴的末尾
    • justify-content: flex-start; 主轴的开始;
    • justify-content: space-around; 元素之间会预留些间隙；
  3. 设置副轴的对其方式
    • align-items: center; 中间
    • align-items: flex-end; 底部
    • align-items: flex-start; 顶部（默认值）
  4. 设置单个元素的副轴的位置
    • align-self: flex-end;
    • align-self: flex-start;
    • align-self: center;

渐变

  • 需要设置给元素的图片属性比如background或者background-image
  • 渐变分为两种
    • 线性渐变—朝着某个方向渐变 background:linear-gradient(to top,green,orange);
    • 径向渐变—圆形 background:linear-gradient(100px,red,yellow);

  • 线性渐变

    • 设置方向用to,可以写具体的角度
    • 颜色后可以加具体的像素，%
    • 例如：background:linear-gradient(45deg,green 40%,red 40%);
    • 线性渐变的具体的代码如下：

  • 径向渐变

    • 括号中先写径向渐变的半径，写一个值是圆，两个值是椭圆
    • 设置方向用at
    • 颜色可以后面可以设置的范围，像素或者%
    • 例如：background:radial-gradient(100px at center,red 50%,orange 50%);

background，border新属性background-size、background-origin、多重背景、border-image

  1. background-size
  • background-size是设置背景图片的大小
  • 属性值有 cover，contain还可以设置具体的像素
  • cover直接等比例放大，直到达到宽或者高较大的一方，出现的问题是可能超出，背景图片显示不完整；
  • contain也是等比例放大，直到达到宽或者高较小的一方就停止，出现的问题是可能不能铺满整个盒子
  • background-size的代码如下：
  1. background-origin
  • 属性值（3个）
    • content-box :背景从内容开始,不考虑 padding的间隙
    • border-box 背景从边框开始
    • padding-box 背景的从内边距开始
  • 复写时直接写在background末尾
  • background-origin的代码如下：
  1. 多重背景
  • css3支持设置多个背景图片，写多个url，用逗号分隔
  • 多重背景一般会在末尾加一个颜色，当图片加载不出来，可以显示颜色，提升用户体验
  1. border-image(兼容性 较差)
  • 实际开发的时候,为了保证 用户起码有一个最基本的相框,会先设置一个 border属性,如果不设置border属性 那么 就无法使用 图片相框,当浏览器不支持 border-image属性时 会使用默认的相框来显示
  • 图片边框 默认的宽度 是 border的宽度

  3. transition-delay 延迟的时间
  4. transition-timing-function 动画的线型
    • ease （默认）
    • easein

transform属性

  1. 移动 transform:translateX(50px) translateY(50px); * 向右是x轴正方向，向下是y轴正方向
    • 也可以复合写：transform:translate(50px,50px);
  2. 旋转 transform: rotate(180deg);
    • 默认是以元素的中心为圆心旋转的
    • transform-origin 可以设置旋转的圆心
      • 可以写具体的位置单词，也可以写具体的坐标
        例如：transform-origin: 20px 20px;
  3. 缩放 transform: scale(2,3);括号中写的是倍数
  4. 扭曲(翻转) transform: skew(45deg,45deg);如果只设置一个值是扭曲，设置两个值是翻转的效果,如果设置两个值相同，则会变得消失
