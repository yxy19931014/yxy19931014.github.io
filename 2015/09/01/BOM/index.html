BOM的小分享

Veröffentlicht am 2015-09-01

BOM

BOM介绍

  • 页面中的顶级对象document
  • 浏览器中的顶级对象window
  • 浏览器包括多个页面，还有其他
  • 页面中的变量和函数，document都是window的
  • 但是有些东西只能document操作，有些只能window操作
  • window在页面中可以省略

    BOM下的方法（window对象中的方法）

  1. window.open(“01祝福墙.html”,”_blank”,”width=300 height=400 resizable=no”);
    • 此方法的第一个参数是地址（可以使用外部的地址，也可以使用本地的地址）
    • 第二个参数是打开方式
      *　第三个参数是打开浏览器的样式还有其他属性
    • 返回值是一个窗口对象（win）
      1. win.close();关闭新打开的窗口
      2. win.moveTo(300,500);让新窗口移动到指定的位置
      3. win.moveBy();每次让新窗口移动多少像素
      4. win.resizeTo(400,500);让新窗口大小改变
      5. win.resizeBy(40,50);让新窗口每次变化多少
  2. location是window下的操作地址栏的对象
    • window.location.href=”http://baidu.com";//可以跳转页面，同时内容可以保存到浏览器的历史记录中，所以浏览器可以前进后退
    • window.location.assign(“http://www.baidu.com");//assign()方法可以跳转，保存历史记录
    • window.location.reload();//刷新
    • window.location.replace(“http://www.baidu.com");//replace()替换地址栏上的地址，不会保存到历史记录中
  3. navigator页面加载后获取客户端浏览器的部分信息
    • window.navigator.userAgent;//客户端浏览器的部分信息
    • window.navigator.platform;//系统信息
  4. history对象操作浏览器的历史记录
    • window.history.forward();//前进
    • window.history.back();//后退
  5. 页面加载的事件
    • window.onload=function(){
      //页面上的内容加载完才能触发
      }
    • window.onunload=function(){
      //页面关闭后触发
      }
    • window.onbeforeunload=function(){
      //页面卸载前触发（页面关闭前触发）
      }
