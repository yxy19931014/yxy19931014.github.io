动画设置

  1. 首先定义动画
    • @keyframes 动画名{
      如果只有初始状态和最后状态
      from{}
      to{}
      }
    • @keyframes 动画名{
      10%{
      }
      50%{
      }
      100%{
      }
      }

  2. 将动画设置给需要使用的元素

    • .类名{
      animation-name: 动画名;
      animation-duration: 持续时间;
      animation-timing-function: 动画的线型;和transition的线型一样
      animation-iteration-count:动画的次数;
      animation-play-state: 动画的播放状态;默认值是running,paused是暂停
      animation-dely:动画的延迟
      animation-fill-mode: forwards;动画结束时保持的状态，一般配合固定次数的动画使用

      • 也可以复合写：
      • animation: 动画名 持续时间 延迟时间…(如果有两个时间，第一个是持续时间，第二个是延迟时间);
        }
  3. 将类名加到需要使用的元素
    • 如果一个元素需要使用多个动画，在animate后面用逗号分隔开。

弹性布局

  1. 开启弹性
    • 给父元素设置display: flex;子元素就可以应用弹性布局
    • 弹性布局开启之后，默认分为两个轴
      • 主轴：默认指的是从左向右—x方向
      • 副轴：默认指的是从上往下—y方向
  2. 设置主轴的对其方式
    • justify-content: space-between;元素之间会预留间隙，但是两边顶格；
    • justify-content: flex-end; 主轴的末尾
    • justify-content: flex-start; 主轴的开始;
    • justify-content: space-around; 元素之间会预留些间隙；
  3. 设置副轴的对其方式
    • align-items: center; 中间
    • align-items: flex-end; 底部
    • align-items: flex-start; 顶部（默认值）
  4. 设置单个元素的副轴的位置
    • align-self: flex-end;
    • align-self: flex-start;
    • align-self: center;
