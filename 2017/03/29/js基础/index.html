    数值类型

  • 简单数据类型（栈）传值
    1. 数字类型
      • 浮点数：存在精度丢失问题，尽量避免浮点数的运算

      • isNAN()方法用于判断是不是数字
    2. 字符串类型
      • 字符串的不可变性
      • 转义字符＼ｂ－－－空格
      • 字符串的拼接
      • 输出带有单引号的字符串
    3. 布尔类型
    4. undefined
      • undefined和任何值运算都为NAN

    • 判断数据类型—typeOf(n) 或者 typeOf n
  • 复杂数据类型（堆）传地址

      • null和数字运算时相当于0
      • undefined==null的结果为true

        数值类型转换

  1. 转换成字符串类型(String)
    • 变量+“ ”或者 变量+任意字符串
    • String()
    • 变量.toString()
  2. 转换成数字类型（Number）
    • 减0，乘1，除1
    • Number() 只要带字母就转换为NAN
    • parseInt() 带字母都自动把字母忽略

  3. 转换为布尔类型
    • 除0，“ ”，undefined,null,NAN都转换为true
    • 将变量转换为布尔类型 —- !!变量
