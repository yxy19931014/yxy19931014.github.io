DOM第一天

DOM介绍（文档对象模型）（文档树模型）

  • 文档
    • html文件可以看成是一个方法
    • xml文件也可以看成是一个文档
    • 页面就是一个文档，整个页面可以看成一个对象
    • 每个页面都有一个顶级对象document
  • API
    • 一套方法
  • 作用
    • 操作页面元素（标签）
    • dom就是对页面元素进行增删改查
  • 节点
    • 页面中的所有元素
    • 标签、文本、属性

    获取元素的方式

  • 根据id获取元素document.getElementById(“id名”);
  • 根据类名获取元素document.getElementsByClassName(“类样式的名字”);得到的是一个数组
  • 根据标签名获取元素document.getElementsByTagName(“标签的名称”);

DOM中给元素设置属性

  • 对象.style.属性=”值”;
  • 对象.className=类样式的名字

    • zIndex的值是数字类型

    节点

  1. 节点分类（对象.nodeType）
    • 元素节点（标签）—1
    • 文本节点———-2
    • 属性节点———-3
  2. 节点名称（对象.nodeName）
    • 元素节点：标签的名字（大写）
    • 文本节点：#text
    • 属性节点：属性的名字
  3. 节点的值（对象.nodeValue）
    • 标签节点：null
    • 文本节点：文本的文本内容
    • 属性： 属性的值
  4. 根节点
    • HTML
  5. 节点的获取和设置
  6. 方法
    • parentNode父级节点
    • parentElement 父级标签
    • childNodes当前元素的子节点
    • children当前元素的子元素
    • firstChild 当前元素的第一个子节点
    • firstElementChild 当前元素的第一个子元素
    • lastChild 当前元素的最后一个子节点
    • lastElementChild 当前元素的最后一个子元素
    • previousSibling当前元素的前一个兄弟节点
    • previousElementSibling当前元素的前一个兄弟元素
    • nextSibling当前元素的后一个兄弟节点
    • nextElementSibling当前元素的后一个兄弟元素
